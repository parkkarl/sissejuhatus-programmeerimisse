# Sissejuhatus programmeerimisse (JavaScripti näidetel)

**Grupp:** TAK25 · **Autor:** Karl Park · **Kuupäev:** 11.06.2026

14-slaidiline esitlus, mis tutvustab programmeerimise põhikontseptsioone JavaScripti näidete abil.
Esitlus on tehtud iseseisva HTML-failina — **töötab brauseris ilma internetita**.

## Failid
| Fail | Sisu |
|------|------|
| `index.html` | Esitlus (14 slaidi). Ava brauseris ja esita täisekraanil. |
| `demo-konsool.md` | Kopeeri-kleebi konsoolisnippetid live-demoks (F12 → Console). |

## Kuidas esitada
1. Ava `index.html` brauseris (topeltklõps failil).
2. Vajuta **F11** (täisekraan).
3. Liigu **← →** noolenuppudega (või klõpsa: parem pool = edasi, vasak = tagasi).
4. Iga mõiste juures näita **live-demo brauseri konsoolis**: F12 → Console → kleebi snippet failist `demo-konsool.md`.

## Kuidas teha PDF (esitamiseks)
Brauseris **Ctrl+P** → sihtkohaks "Salvesta PDF-ina" → iga slaid tuleb omaette lehele.
Salvesta nimega: `Sissejuhatus-programmeerimisse-KarlPark.pdf`

## Slaidide kava
1. Tiitel (autor, aine, kuupäev)
2. Mis on JavaScript (+ erinevus HTML/CSS-ist)
3. Arenduskeskkond — kuidas koodi käivitada
4. Andmetüübid (number, string, boolean)
5. Muutujad (let, const)
6. Tehted (aritmeetilised, võrdlus, loogika)
7. Tingimused (if / else)
8. Kordused (for, while)
9. Funktsioonid (+ noolfunktsioon)
10. Massiivid (indeks, length, läbimine)
11. Andmestruktuurid (massiiv, objekt)
12. Sisend ja väljund (console.log, alert, prompt)
13. Algoritm · Klass · Kommentaarid
14. Kokkuvõte — mida ma õppisin
