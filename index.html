Mis on JavaScript?

Programmeerimine = arvutile sammhaaval juhiste andmine, et ta midagi teeks.

JavaScript (JS) on programmeerimiskeel, mis töötab veebibrauseris ja ka serveris (Node.js). Kasutatakse: veebilehtede loogika, serverid, rakendused.

  • HTML — lehe sisu ja struktuur (mis on lehel)
  • CSS — lehe välimus (kuidas asjad välja näevad)
  • JavaScript — lehe käitumine ja loogika (mis juhtub, kui vajutad nuppu)

Arenduskeskkond — kuidas koodi käivitada

Mina kasutan brauseri DevTools konsooli (klahv F12 → vahekaart Console) ja koodi kirjutamiseks VS Code-i.

1) Konsoolis: ava F12 → kirjuta käsk → Enter.

console.log("Tere konsoolist!");

2) HTML-failis <script> abil:

<script>
  console.log("Tere lehelt!");
</script>

Andmetüübid

Andmetüüp ütleb, mis liiki väärtus on — number, tekst või tõeväärtus. Tüüp määrab, mida väärtusega teha tohib.

let täisarv = 42;      // number
let tekst = "Tere";    // string (tekst)
let tõene = true;      // boolean (tõene/väär)

typeof täisarv;        // "number"

Muutujad

Muutuja on nimeline koht mälus, kuhu hoiame väärtuse. let = muudetav, const = konstant (ei muutu).

let vanus = 18;     // muudetav
const PI = 3.14;    // konstant

vanus = 19;         // ✅ lubatud
// PI = 4;          // ❌ annab vea

Tehted

Aritmeetilised

let a = 5, b = 2;
a + b;   // 7   liitmine
a / b;   // 2.5 jagamine
a % b;   // 1   jääk

Võrdlus ja loogika

5 > 3;          // true
5 === 5;        // true (täpne =)
true && false;   // false (JA)

Tingimused

Programm valib, mida teha vastavalt olukorrale: if / else if / else.

let vanus = 17;

if (vanus >= 18) {
  console.log("Täisealine");
} else {
  console.log("Alaealine");   // see käivitub
}

Kordused (tsüklid)

Korduse abil teeme sama tegevust mitu korda, ilma et peaks koodi mitu korda kirjutama.

// for-tsükkel
for (let i = 0; i < 5; i++) {
  console.log("i = " + i);
}
// while-tsükkel
let x = 0;
while (x < 3) {
  console.log("x = " + x);
  x++;
}

Funktsioonid

Funktsioon on taaskasutatav koodiblokk: kirjeldad korra, kutsud välja mitu korda.

function tervita(nimi) {
  console.log("Tere, " + nimi + "!");
}
tervita("Mari");
tervita("Juhan");

// noolfunktsioon (sama mõte, lühem)
const korrutaKahega = (x) => x * 2;
korrutaKahega(5);   // 10

Massiivid

Massiiv on järjestatud elementide kogum. Indeks algab 0-st.

let õpilased = ["Mari", "Juhan", "Kati"];
õpilased[0];       // "Mari"
õpilased.length;   // 3

// massiivi läbimine
for (let i = 0; i < õpilased.length; i++) {
  console.log(õpilased[i]);
}

Andmestruktuurid

Andmestruktuur on viis andmete organiseerimiseks. Massiiv on üks, objekt teine. Objekt hoiab nimi–väärtus paare.

let arvud = [1, 2, 3];                     // massiiv
let inimene = { nimi: "Mari", vanus: 20 }; // objekt

inimene.nimi;    // "Mari"
inimene.vanus;   // 20

Sisend ja väljund

Väljund (programm näitab midagi)

console.log("Tere maailm!"); // konsooli
alert("Hoiatusaken!");       // hüpikaken

Sisend (kasutaja annab midagi)

let nimi = prompt("Sisesta nimi:");
console.log("Tere, " + nimi + "!");

Päris elus käib sisend/väljund tihti vormide, failide ja API-de kaudu — praegu piisab nendest.

Algoritm · Klass · Kommentaarid

  • Algoritm — täpne sammude jada probleemi lahendamiseks (nagu retsept).
  • Klass — mall ehk vorm sarnaste objektide loomiseks.
  • Kommentaarid — selgitused koodis, mida arvuti ei käivita.
class Auto {
  constructor(mark, aasta) {
    this.mark = mark;
    this.aasta = aasta;
  }
}
let a = new Auto("Toyota", 2020);

// üherealine kommentaar
/* mitmerealine
   kommentaar */

Kokkuvõte — mida ma õppisin

  • Tean, mis on muutuja, andmetüüp, massiiv, objekt, funktsioon, tingimus ja kordus.
  • Oskan lugeda lihtsat JavaScripti koodi ja aru saada, mis see teeb.
  • Oskan koodi käivitada brauseri konsoolis (F12 → Console) ja <script> abil.
  • Saan aru, et andmetüübid ja struktuurid aitavad andmeid korras hoida.
